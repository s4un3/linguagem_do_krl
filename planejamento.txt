variaveis e funções declaradas com "var" e "func"
variáveis locais com "loc"
variáveis criadas dentro de funções são automaticamente deletadas, mas devem ter nomes diferentes às externas para não causar conflito

comparação de strings com "==" e nada de "strcmp" ou coisa parecida

convenções: nomes de variáveis internas a funções começam com "_"

exemplos:

var int x = in;
var int y;

func quadrado(int _x) -> int{
    new int _y = _x**2;
    return _y;
}
y=quadrado(x);

clear(x);
clear(y);
